Mermaidly Demo

Welcome to Mermaidly! This document demonstrates the key features of the markdown renderer with embedded Mermaid diagrams.

Quick Start

Mermaidly makes it easy to include diagrams in your markdown documentation. Just use fenced code blocks with the mermaid language identifier.

Features

Zoomable Diagrams

All diagrams support:

  • Mouse wheel - Zoom in/out
  • Click and drag - Pan around
  • Zoom buttons - +/- controls
  • Reset button - Return to original view

Table of Contents

The sidebar shows a clickable table of contents generated from your markdown headings. Click any heading to smoothly scroll to that section.

Themes

Mermaidly supports multiple themes:

  • default - Light theme
  • dark - Dark theme
  • forest - Green theme
  • neutral - Grayscale theme

Export

Export your diagrams as PNG images with configurable scale:

mermaidly export demo.md --scale 2 --output-dir ./images

Conclusion

Mermaidly makes it easy to create beautiful, interactive documentation with embedded diagrams. Try it out!
